About me

I have been programming since I was 9. I started by creating games using QBASIC, and upgraded to QB64 a year or so later. After that I dove into Python and C++ for a while. After experimenting with lots of languages and frameworks, I decided to give Lua and LÖVE a try. After a year or so in that community, I started working with Binary Cocoa, working on a couple games. After a few months we released HDF (Hexapod Defense Force). All of the game logic was written in Lua. I wrote the savefile system, and implemented a few of the critters in-game. Now I spend my time learning about different programming languages, and how to implement them.

Some of my projects

PHPBF

Brainfuck is an esoteric, minimalist programming language created by Urban Müller in 1993. I say minimalist because it only has 8 commands (>, <, +, -, ., ,, [, ,]). I created PHPBF so that I would have the option of compiling, and or interpretering Brainfuck code while I set out to learn the language. Visit the PHPBF page here.

Derplang

Derplang is the first programming language I designed, and implemented from scratch. It started as an esoteric programming language, but because of interest from a few members of my IRC channel, I decided to redesign/reimplement it. We are still working on Derplang to this day, and if you would like to check it out, visit the Derplang page here, or join #derplang on Freenode.
